# Amigo Secreto - Curso Java Script

Este projeto implementa o back-end de uma aplicação de “Amigo Secreto” onde usuários podem:
- cadastrar nomes
- visualizar a lista de nomes
- sortear aleatoriamente um amigohttps://github.com/DaiMonteiro1010/TESTE-AMIGO-SECRETO/tree/main

O armazenamento é feito em memória e pode ser facilmente conectado a um banco de dados posteriormente.
